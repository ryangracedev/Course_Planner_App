REST API Documentation

List All Courses

GEThttps://cis3760f23-10.socs.uoguelph.ca/rest/api.php?course=all

Description

Returns a list of all courses available. The data returned for each course is:

Field Value Type
subject_area The abbriviated name for the subject area the course belongs to String
number The unique id for the course in it's subject area. Larger numbers indicate more advanced courses Integer
name The display name of the course String
weight The number of credits granted for completing the course Float
description The text details of the course including what material is taught String
department The name of the academic department that administrates the course String
location Where the course is taught String
prerequisites The prerequisites of the course in a textual representation String
credits_required The number of credits needed to take the course Float

Search Course

GEThttps://cis3760f23-10.socs.uoguelph.ca/rest/api.php?course=<course_code>

Query Parameters

Field Value
course_code The course code to search. Should be of the format [subject_area]*[course_number]

Description

Returns a list of courses that match the specified course. If the course does not match anything, an empty list is returned. The data returned is:

Field Value Type
subject_area The abbriviated name for the subject area the course belongs to String
number The unique id for the course in it's subject area. Larger numbers indicate more advanced courses Integer
name The display name of the course String
weight The number of credits granted for completing the course Float
description The text details of the course including what material is taught String
department The name of the academic department that administrates the course String
location Where the course is taught String
prerequisites The prerequisites of the course in a textual representation String
credits_required The number of credits needed to take the course Float

List All Prerequisite Courses

GEThttps://cis3760f23-10.socs.uoguelph.ca/rest/api.php?prereq=<course_code>

Description

Returns a list of all available subject codes. The data returned is:

Field Value Type
id The id of the group the requirement belongs to Integer
parent_id The id of the parent group the requirement is a child of. If null this is the root node of the requirements tree Integer | Null
num_required The number of requirements that must be satisfied in this group plus the number of children groups that must be satisfied Integer
subject_area The course subject area of the requirement String | Null
number The course number of the requirement Integer | Null
name The course name of the requirement String | Null

List All Qualified Courses

POSThttps://cis3760f23-10.socs.uoguelph.ca/rest/api.php

Request Body

The body of the request should only contain the courses that have already been taken. Each course should use the key course<id> where id is a unique number for each course

Example:

This command can be executed in the browser's console

                
    fetch("https://cis3760f23-10.socs.uoguelph.ca/rest/api.php", {
        method: "POST",
        body: JSON.stringify({
            "course1": "CIS*2750", "course2": "CIS*3750"
        }),
        headers: {
            "Content-type": "application/json; charset=UTF-8"
        }
    })

Description

Returns a list of all available subject codes that the user can take given the courses input. The data returned is:

Field Value Type
subject_area The abbriviated name for the subject area the course belongs to String
number The unique id for the course in it's subject area. Larger numbers indicate more advanced courses Integer
name The display name of the course String
weight The number of credits granted for completing the course Float
description The text details of the course including what material is taught String
department The name of the academic department that administrates the course String
location Where the course is taught String
prerequisites The prerequisites of the course in a textual representation String
credits_required The number of credits needed to take the course Float
